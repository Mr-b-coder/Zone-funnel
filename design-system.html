Spacing & Grid

We use an 8-Point Grid System. All spacing and component dimensions are multiples of 8px, ensuring visual consistency and rhythm.

// Tailwind Configuration

theme: {

  spacing: {

    '1': '8px',

    '2': '16px',

    '3': '24px',

    '4': '32px',

    '5': '40px',

    '6': '48px',

    // etc.

  }

}
